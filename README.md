# <beverage>

## サイト概要
### サイトテーマ
自分の好みの飲み物を探せたり、飲み物について投稿できるサイト

### テーマを選んだ理由
もともと新商品の飲み物を飲んだり、いろいろな飲み物を飲んだりしていましたが、好みに合わなかったり、
失敗したりする事がありました。事前に商品の情報や評価が分かれば便利だと考え、このテーマにしました。
### ターゲットユーザ
飲み物が好きな人

### 主な利用シーン
飲みたい飲み物を検索する時、飲んだ飲み物を投稿する時
