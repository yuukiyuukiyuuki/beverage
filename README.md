# beverage

## サイト概要
### サイトテーマ
自分の好みの飲み物を探せたり、飲み物について投稿できるサイト

### テーマを選んだ理由
もともと新商品の飲み物を飲んだり、いろいろな飲み物を飲んだりしていましたが、好みに合わなかったり、  　
失敗したりする事がありました。夏の時期になると飲み物の消費量も増えるので、  
事前に商品の情報や評価が分かれば便利だと考え、このテーマにしました。
### ターゲットユーザ
- 飲み物が好きな人
- 飲み物に興味がある人
- 普段からよく飲み物を買う人
- 同じ価値観に共有したい人
- いろんな飲み物の情報が知りたい人

### 主な利用シーン
- 飲みたい飲み物を検索する時
- 飲んだ飲み物を投稿する時
- 投稿したユーザを検索する時
- 好きな投稿に対してブックマークする時
- 知りたい情報に対してフォローする時
- 投稿に対してコメントする時
- 飲み物に対してレビューする時
